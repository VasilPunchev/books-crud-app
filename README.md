# 📚 Books CRUD App

A simple JavaScript CRUD application for managing books via REST API.

## 🚀 Features

- 📖 Load all books
- ➕ Create new book
- ✏️ Edit existing book
- ❌ Delete book
- 🔄 Dynamic table updates
- ⚡ Async/Await with Fetch API

## 🛠 Tech Stack

- JavaScript (ES6)
- Fetch API
- HTML5
- CSS3
- REST API (JSONStore)
